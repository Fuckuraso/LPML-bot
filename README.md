# LPML-bot
Bot for education server in discord                           
How to use:                                        
.p - відмітити присутність учня                       
.pn - список відсутніх                        
.v a1 a2 - відмітка учневі, де a1 - це @нік, а a2 - це тип відмітки(S, extra, |, ...)  
.va - всі відмітки за урок
